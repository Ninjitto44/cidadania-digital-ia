# Cidadania Digital e Inteligência Artificial

## Objetivo do Projeto
Este site foi desenvolvido como parte da Recuperação Prática da disciplina de Educação Digital e IA. O objetivo é conscientizar a comunidade escolar sobre os perigos das deepfakes e da desinformação, trazendo dicas de como identificar mídias manipuladas e testar o conhecimento do usuário por meio de um quiz interativo.

## Tecnologias Utilizadas
- HTML5 (Estrutura Semântica)
- CSS3 (Flexbox e Responsividade)
- JavaScript Vanilla (Manipulação de DOM)
